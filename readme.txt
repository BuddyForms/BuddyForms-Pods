Use BuddyForms with Pods

== Description ==

With the BuddyForms Pods extension, you bring the potential from Pods and BuddyForms together. Create outstanding Pods and use BuddyForms for content creation and management.

You can make any part of your Pods accessible in your BuddyForms forms



    Use any Pods type with BuddyForms
    Use Pods in any Form
    Use a single Pods field from any Pods
    Use Pods fieldsets from any Pods type



Create frontend forms for any Pods with moderation control and all the tools for content creation provided by BuddyForms with this extension.

Find all information under: https://themekraft.com/products/buddyforms-pods/
